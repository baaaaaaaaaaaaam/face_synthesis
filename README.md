# 팀플 포트폴리오 -3

# 움짤 제작소

영상속 재미있는 장면에 자신의 얼굴과 녹음한 음성을 합성하여 영상으로 만들고 다운받을수 있는 앱입니다.

# 기능
1. 아이디 중복 체크
2. 이미지 업로드
3. 음성 녹음
4. 영상 합성
5. 영상 다운로드


# 역활

팀장 및 역활 분배

서버 구축

# 사용기술

● 서버: GCP(Google Cloud Platform)

● 웹 서버: Nginx, uwsgi, flask

● 데이터베이스: Mysql

● 언어: Java, Python

● 라이브러리/API : OpenCV, dlib , FFmpeg
